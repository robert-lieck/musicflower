musicflower.loader.audio_scape

musicflower.loader.audio_scape(n_time_intervals, data=None, raw_chroma=None, normalise=True, top_down=False, **kwargs)[source]

Compute a pitch scape from audio data.

Parameters:

  • n_time_intervals (int) – number of time intervals

  • data (Union[str, Tuple[ndarray, int], None]) – path to file or a tuple with audio data (passed on to get_chroma())

  • raw_chroma (Optional[ndarray]) – pre-extracted raw chroma (ignore data)

  • normalise (bool) – normalise the pitch class distributions

  • top_down (bool) – use top-down order

  • kwargs – kwargs passed on to the get_chroma() function

Return type:

ndarray

Returns:

array with pitch scape
